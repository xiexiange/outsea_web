中国などでは **HD17** と呼ばれるモデルが、**Dyson Supersonic r™** ヘアドライヤーに相当します。従来の Supersonic™ より小さく軽く、自宅で「速く乾かして、少し整える」用途向け。海外 EC で比較する方向けに、**髪質と使い方への適合**だけ整理します。

## 向いている人

- **セミロング以上**を週に何度か自分で乾かす
- **細い髪・うねり**が気になり、マイナスイオンや冷風で仕上げたい
- **複数ノズル**（速乾・やさしい風・なめらか・集中）を使い分ける意思がある
- 最安ワット数より **温度管理・モーター・再販価値**を重視

向いていにくい例：とにかく高温最速だけ欲しい、予算が極端に低い、**巻き髪メイン**（Airwrap の領域）。

## 比較ポイント

| 項目 | 意味 |
|------|------|
| **型番** | **HD17 ≒ Supersonic r™**、**HD16 ≒ Nural™**、丸型は Supersonic™ |
| **重量** | r™ は従来ホスト比 **30%小型・20%軽量**（公式比較）の訴求が多い |
| **電圧** | **230V / 1700W** などが多い—購入国の電圧・保証を確認 |
| **RFID ノズル** | 5 種のスマートノズルが設定を記憶 |
| **フィルター** | 柄付近のフィルター清掃が必要。黄ランプでお知らせ |

::: tip
**軽量ドライヤー（HD17 / r™）**か**巻き髪マルチ（Airwrap）**かを先に決めましょう。役割が違います。
:::

## モデルのざっくり地図

**本命：HD17 / Supersonic r™** — 流線型ヒーター、Hyperdymium™ モーター、5 ノズル。日常の乾燥＋軽いスタイリング。

**頭皮寄り：HD16 / Nural™** — 頭皮モードなど。近距離で乾かす方に。

**従来：Supersonic™** — 付属品が豊富、価格が下がることも。

**別ジャンル：Airwrap** — ドライヤー代替ではない。

## 避けたい買い方

- ほぼ乾かさないのにフルセット
- **保証なし並行品・電圧不一致**
- フィルター未清掃のまま使用
- 巻き髪が主目的なのに HD17 のみ購入

## 購入前チェック

1. **型番・同梱ノズル**を完全一致で確認  
2. **電圧・プラグ・保証**  
3. **直近の低評価**（熱停止・騒音・サポート）  
4. Nural・従来 Supersonic・他社と**総額**比較  
5. Amazon の価格・在庫は変動—**型番と同梱**を購入前に確認
